# El Mirasol — Mexican Grill & Store

Static website for El Mirasol in Burgaw, NC.

## Live URLs (after GitHub Pages is enabled)

- Home: `https://YOUR-USERNAME.github.io/el-mirasol/`
- Menu: `https://YOUR-USERNAME.github.io/el-mirasol/menu.html`
- Google Sites embed: add `?embed=1` to either URL

## GitHub Pages setup

1. Push this folder to a GitHub repo named `el-mirasol`
2. Repo **Settings** → **Pages**
3. **Build and deployment** → Source: **Deploy from a branch**
4. Branch: **main** → folder **/ (root)** → Save
5. Wait 1–2 minutes, then open the site URL above
